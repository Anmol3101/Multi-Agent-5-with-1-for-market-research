# Multi-Agent-5-with-1-for-market-research


HyperMarket 5+1 is a multi‑agent market research engine that turns any product + market into a full strategy pack in one shot. It runs five specialized ‘agents’ (research, strategy, numeric verifier, logic verifier, resolver) plus a master writer to produce a structured JSON dataset, a ready‑to‑fill Excel workbook spec, and a slide‑deck spec. Designed for token‑efficient deep dives, it dynamically adjusts depth (simple/standard/complex) and self‑evaluates each run to suggest how to improve future analyses.
